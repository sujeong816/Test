프론트엔드 사용자 인터페이스 담당

1. 데이터를 잘 사용자에게 보여주기

2. 데이터 조작(화면 조작)
-현재 화면 데이터로 구성
-새로운 데이터 생성
-사용자의 이벤트 감지 및 분류
-생성된 데이터 화면이 출력
-행동에 맞는 적절한 api 동작

3. 서버에 데이터 보내기
-api 문서 숙지
-서버에서 전의한 스키마에 맞게 내부적으로 타입 설정
-화면에 있는 데이터를 api에서 사용하는 구조로 변경

4. 서버에서 받은 데이터 다루기
-서버에 있는 데이터를 전달받아 화면에 필요한 데이터로 전화하여 변환하는 과정
-비동기성 특징으로 인한 로딩 중복방지 등 처리
